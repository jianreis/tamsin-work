# Summary

MomConnect (launched 2014) is the proof point that **multi-channel,
low-bandwidth health messaging works at national scale in SA**: USSD-based
registration on feature phones (no data), stage-based SMS, and a later **WhatsApp
chatbot** (HealthConnect). The reference pattern for reaching the full SA
population — feature-phone and smartphone alike — for a scripting service.

# Key claims
- MomConnect reached **~5 million registered mothers** and operated across **>95%
  of public health facilities** nationally — demonstrating feature-phone-first
  health services scale in SA.
- **USSD registration** lets feature-phone users enrol by dialling a short code —
  **no smartphone, no data cost** — capturing details at point of care.
- Stage-based messaging delivered via **SMS and later WhatsApp**; the WhatsApp
  migration (Praekelt, ~2017) added richer, two-way, interactive messaging
  (including photos) via WhatsApp's Business API + chatbot.
- The model deliberately spans channels to **meet users at their device
  capability** rather than forcing an app — a template for inclusive, layered
  channel design.
- SA government and commerce widely use **WhatsApp** as a primary service channel
  (e.g. COVID-19 helpline in multiple official languages); WhatsApp Business API
  supports **all 11 official languages**.

# Verbatim excerpts
> "Expectant mothers dial a USSD code at their first antenatal visit… No
